
# enigma-rust

An implementation of the Enigma Machine in Rust

## Run Locally

Clone the project

```bash
  git clone https://github.com/corslyn/enigma-rust
```

Go to the project directory

```bash
  cd enigma-rust
```

Compile and start the program

```bash
  cargo run
```
